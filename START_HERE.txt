GestureFlow Park 启动说明

1. 启动后端
cd D:\project\gestureflow_hci_project_v4\backend
npm install --registry=https://registry.npmmirror.com/
npm run dev

2. 启动前端
cd D:\project\gestureflow_hci_project_v4\frontend
npm install --registry=https://registry.npmmirror.com/
npm run dev
